# 3D Perspective

The other day I found this tweet from [@CodyWebHouse](https://twitter.com/CodyWebHouse)

<TwitterTweetEmbed
  tweetId={'1326916915479142401'}
/>

And wanted to try it out myself.

I created a ProfilePicture component:

```js
export default function ProfilePicture () {
  return (
    <figure className={styles.imageContainer}>
      <Image
        src="/images/profile.png"
        alt="m4nu56"
        layout="fill"
        quality={100}
      />
    </figure>
  )
}
```

And applied the same CSS rules as indicated in the tweet:

```scss
.figureCentered {
  @apply text-center;
}

.imageContainer {
  @apply w-56 h-56;
  perspective: 1500px;
  @apply mx-auto;
  img {
    @apply rounded-full;
    transform: rotate(0);
    transition: .3s;
  }

  &:hover img {
    transform: rotateX(15deg) rotateY(-18deg) rotateZ(3deg);
  }
}
```

And Tada! Try and move your mouse over the image to see it rotate to the position I indicated in the transition
