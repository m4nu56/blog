# Java Optionals

Replace `if != null` statement with a Java optional

```java
if(myVar != null){
    logger.info("MyVar=" + myVar);
}
```

```java
Optional.ofNullable(myVar).ifPresent(var -> logger.info("MyVar=" + var));
```
